# Gym Management System  (Versión SQL)

Este es un sistema web sencillo (CRUD) para gestionar entrenamientos y ejercicios usando **PHP**, **MySQL** y **Docker**.

##  Características

-  Crear, editar y eliminar sesiones de entrenamiento.
-  Agregar, editar y eliminar ejercicios.
-  Registrar los detalles de cada entrenamiento (series, repeticiones, peso).
-  Visualización organizada de entrenamientos y sus ejercicios.

## Tecnologías utilizadas

- PHP
- MySQL
- Docker
- HTML / CSS
